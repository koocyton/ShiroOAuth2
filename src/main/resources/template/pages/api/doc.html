软件功能分类：

一.个人信息
1. 关注人：可以查看关注自己的人。
2. 基本信息：包含昵称，用户ID，用户性别，年龄及星座（其中星座为计算出来）。
3. 座驾：
4. 编辑功能
* 上传照片，可以上传照片，单次可上传9张，最大可上传，照片可以通过按住移动编辑显示顺序。
* 个人签名，修改后连同服务器立即生效。
* 昵称，最大不能超过16个字。
* 生日，显示用户的生日，同理用于显示用户星座。
* 兴趣爱好，默认显示为空，需要用户手动输入。
* 身高，默认显示为空，需要用户手动输入。
* 常住地，默认显示为空，需要用户手动输入。
* 聊天对象，默认选中都可以，点击后弹出下拉菜单，内容包含：男生，女生，都可以。
* 属性，默认为空，需要用户手动选择，最多三个，所有标签为系统预设，不能手动编辑。
5.  座驾
* 钻石购买，最多只显示1辆座驾。座驾有时间限制。
* 点击试驾，特效显示该汽车效果特效。每次试驾显示一次。
* 车库，显示用户自身车库中的所有汽车。可以在此处编辑展示的车辆。

二.最新上线
1. 定义：房间id，每一个房间都有自己独立的房间id，当房间删除时候，id也删除，可被重复使用。
2. 房间，显示所有最新上线的房间列表，最大显示数量暂定。排序根据房间创建时间排序。
* 进入我的房间，创建自己的房间，自己为房主
       A.房主权限1:报用户上麦，邀请旁听的用户上麦位。
       B.房主权限2:封闭此麦位，此麦位不能有用户上位。
       C.房主权限3:禁麦，该麦位用户不能发言。
       D.房主权限4:打开音乐权限，音乐权限为该用户是否可以播放音乐。
* 房间操作按钮（右上角）
       A.退出房间，离开该房间，也不会最小化该房间信息。
       B.邀请好友，一键分享到微信或QQ。
       C.修改房间名，可以修改房间的名字。
       D.房间加锁，为房间设置密码，其他人加入需要密码。
       E.关闭公屏，关闭系统频道消息。
       F.举报房间，举报该房间（只存在于游客进入时候显示）
* 听筒和麦克风，关闭听筒或麦克风，则不能语音或听到其他人信息。
* 其他按钮
       A.主题，房间的背景风格。
       B.音乐，我的曲库为我下载的音乐，热门音乐中可以通过搜索查找音乐。
       C.幸福瞬间，给主播的礼物。
       D.小纸条，给在位的用户发送文字消息。
       E.管理员，添加房间内的管理员，管理员权限：禁麦，封麦，踢人，抱人。管理员可进行编辑。
       F.表情，发送表情信息。
       G.文字，发送文字信息。
  3.  附近，显示附近的房间列表，根据房间创建时间排序，需要用户开启定位服务。
  4.  热门
* 热门房间，显示房间围观人数最多的9个房间。
* 推荐用户，随机显示20个用户，可以通过换一批更换推荐用户显示用户逻辑待定。
  5.  搜索
* 找房间
       A.通过关键词模糊搜索。显示用户名（房间名）中包含该关键字的房间列表。
       B.最近热搜，系统根据搜索次数，给出关键字被搜次数最高的前10个，此规则暂定。
       C.搜索历史，显示用户的搜索关键字历史，可清除。
* 找朋友
       A.根据Id或者用户昵称搜索。区别搜索房间，该搜索只显示用户名的用户。
       B.条件查找，根据条件批量搜索用户。

三.我的账户
1. 用于显示用户的钻石和金币。
2. 钻石获得途径：
* 充值，根据配置的充值金额获得。
* 任务，完成新手任务获得。
3.  金币获取途径：
* 根据设置的任务，完成后可以一次性获得奖励。
4.消息
  1.  通话记录
  2.  消息记录
* 系统消息，系统推送的消息。
  A.兑换和提现，根据兑换和提现问题的处理。
  B.玩转hello，关于充值问题或者功能使用问题的解释。
  C.社区规范，包含封号申诉和社区规范解释。
* 其他人消息，其他人发的消息，例如：纸条。
5.联系人
  1.  好友，对方同意添加的好友。
  2.  粉丝，个人信息中点击关注即可成为粉丝。
6.活动排行榜
  1.  总榜，所有用户的魅力值或贡献值的总排行榜。
  2.  鬼火，活动排行榜。
7.活动中心
1. 显示所有的正在进行，已经结束的活动的活动。
8.任务奖励
1. 完成任务可获得金币奖励。

用户体验不足之处
1. 购买物品时候不显示钻石或金币
