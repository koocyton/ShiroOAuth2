# 登录 #
API URL : [POST] http://domain.com/api/v1/login
POST 参数 :
account : #account
password : #password
返回 :
json : {#accesstoken}
测试返回 :
json : #右侧测试结果显示在这#
登录的数据会保存，其他需要 access token 的接口，会直接用最近一次登录的 access token
